# Agent Builder Proposal

Load this skill only when the `.md` drafting or refinement proposal is ready to present.

## Required Actions

- Present the planned `.md` changes clearly.
- For updates to existing files, show the actual inline diff in chat.
- Ask for explicit approval before writing.
- If feedback changes scope, revise the proposal and ask for approval again.

## Limits

- Do not write files before explicit approval.
- Do not omit the inline diff when proposing updates to existing files.
