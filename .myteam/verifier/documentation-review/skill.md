# Verifier Documentation Review

Load this skill only when evaluating whether updated documentation accurately reflects the shipped behavior.

## Required Actions

- Determine whether updated documentation matches the implemented and tested behavior.
- Identify missing documentation updates, inaccurate statements, contradictory details, or duplicated facts across docs.
- Treat materially inaccurate or incomplete documentation as a review finding when it creates delivery or maintenance risk.
