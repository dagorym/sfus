## Context

ST5 adds per-board Topics, Posts, and Last Post columns to the public /forums
index page. The board list previously rendered as a <ul>; it now renders as a
semantic <table> with four columns: Board, Topics, Posts, and Last Post.
PublicBoardShape in forums-client.ts gained topicCount, postCount, and
lastPost (BoardLastPostShape | null). Last Post renders an absolute date via
toLocaleDateString() and a profile link (displayName ?? username, username
encodeURIComponent-encoded) to /users/<username>; "No posts yet" when
lastPost is null. No dangerouslySetInnerHTML is used anywhere in the index page.
