### Key behaviors

- `getDocPageTree(parentPath?)` fetches GET /api/docs; returns null on 404, throws on other errors. Uses extractErrorMessage(payload, fallback) which reads payload.error?.message || payload.message || fallback.
- `getDocPageByPath(path)` fetches GET /api/docs/<encoded-path>; returns null on 404, throws on other errors.
- `getRecentDocEdits(limit?)` fetches GET /api/docs/recent; never returns null, throws on error.
- All public read routes use `cache: "no-store"` and do not include `credentials: "include"`.
- Lock state: `isLocked = page.lock.isLocked && page.lock.lockExpiresAt !== null && new Date(lockExpiresAt) > new Date()`.
- The /docs/new and /docs/<path>/edit routes do NOT exist yet (ST-8 scope) — staff affordance links point to those not-yet-existing routes. This is expected and intentional.
- Both route files export only `export default function`.
